# Corax Arcana

**Corax Arcana** é um projeto educacional completo de **Engenharia de Dados**, criado por um iniciante com o objetivo de praticar todas as etapas de um pipeline moderno, utilizando ferramentas de mercado e seguindo boas práticas.

---

## Objetivo do Projeto

Criar um **pipeline de dados completo** para:

- Coletar cartas do jogo *Magic: The Gathering* usando a **API pública da Scryfall**.
- Orquestrar o fluxo de dados com **Apache Airflow**.
- Armazenar os dados em um **Data Lakehouse com DuckDB**, seguindo a **arquitetura Medallion** (Bronze, Silver, Gold).
- Transformar os dados com **DBT (Data Build Tool)**.
- Visualizar os dados com **Streamlit**, criando dashboards e análises interativas.
- Empacotar tudo com **Docker** e publicar em ambientes controlados usando o **Cosmos da Astronomer**.
- Manter o projeto versionado e organizado com **Git e GitHub**.
